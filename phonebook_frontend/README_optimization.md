# Phonebook 应用优化版本

## 文件说明

- `src/App.jsx` - 原始版本
- `src/App_better.jsx` - 优化版本
- `src/components/better_files.md` - 详细对比文档

## 如何切换版本

### 使用原始版本
```javascript
// 在 main.jsx 中
import App from './App.jsx'
```

### 使用优化版本
```javascript
// 在 main.jsx 中
import App from './App_better.jsx'
```
